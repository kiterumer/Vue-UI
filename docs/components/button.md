# Button - 按钮

## 简单用法
**预览**
<ClientOnly>
  <button-demo></button-demo>
</ClientOnly>

**代码**
```html
<g-button>默认按钮</g-button>
<g-button icon="settings">默认按钮</g-button>
<g-button :loading="true">默认按钮</g-button>
<g-button disabled>默认按钮</g-button>
```
